Check out the trunk version of the project from the URL::

    svn checkout svn://svn.binaryvibes.co.in/bizsense/trunk to/this/directory --username=yourusername

You can omit the username switch in the above command, if you do not wish to commit to the repository.

To view the repository in a web browser visit http://projects.binaryvibes.co.in/repositories/show/bizsense
