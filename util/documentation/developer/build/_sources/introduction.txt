Introduction
--------------------

Bizsense project adheres to the coding standards for PHP and JavaScript. We also follow naming conventions for HTML, CSS and MySQL.

The project makes use of the follwing:

* PHP, Zend Framework, Bare Framework
* MySQL
* JavaScript, Dojo Toolkit
* XHTML 1 and CSS 2

In order to contribute code to the Bizsense project, you have to be familiar with the above technologies.
